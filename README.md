🗺️ Карта путешествий (Map)

Интерактивная карта мира для отметки посещённых стран и планирования будущих поездок.
Вы можете присвоить каждой стране одну из четырёх категорий, и она будет закрашена соответствующим цветом. Все данные сохраняются в вашем браузере (localStorage).
✨ Возможности

    🖱️ Клик по стране — открывает модальное окно с выбором категории.

    🎨 4 категории с уникальными цветами:

        ✅ Посетил — зелёный

        ⭐ Хочу посетить — жёлтый

        🛂 Жду визу — оранжевый

        🚫 Вряд ли посещу — красный

    📋 Список отмеченных стран справа с цветными индикаторами.

    🗑️ Удаление отдельной страны прямо из списка.

    🔄 Сброс всех меток одной кнопкой.

    💾 Автоматическое сохранение в localStorage — после перезагрузки страницы все ваши отметки остаются.

    🌍 Карта на основе Leaflet с подложкой OpenStreetMap (названия стран и городов видны всегда).

    📦 Полностью локальная работа — все библиотеки и данные (GeoJSON) лежат рядом с HTML, не требуется интернет для отображения границ (но подложка OSM загружается с сети).

🚀 Как развернуть
Вариант 1: GitHub Pages (рекомендуется)

    Скачайте или склонируйте этот репозиторий.

    Убедитесь, что все файлы лежат в корневой папке (см. структуру выше).

    Загрузите эту папку в новый репозиторий на GitHub.

    В настройках репозитория (Settings → Pages) выберите ветку main (или master) и папку /root, сохраните.

    Через пару минут ваш сайт будет доступен по адресу https://ваш-логин.github.io/имя-репозитория.

Вариант 2: Локальный запуск без сервера (с ограничениями)

Из-за политики безопасности браузеров (CORS) при открытии index.html напрямую может не загрузиться файл countries.geojson.
Чтобы всё работало локально, используйте любой простой веб-сервер:

    Python 3: выполните в папке проекта python -m http.server, затем откройте http://localhost:8000.

    VS Code: установите расширение Live Server и запустите через него.

    Mongoose (Windows): скачайте mongoose.exe, положите в папку и запустите.

Вариант 3: Скачать все зависимости вручную

Если вы хотите полную автономность (без интернета), скачайте тайлы OpenStreetMap — но это сложно. Для большинства случаев достаточно локальных библиотек и GeoJSON.
🛠️ Использование

    Кликните по любой стране на карте.

    В появившемся окне выберите одну из категорий (или "Убрать метку", чтобы удалить цвет).

    Страна сразу перекрасится в соответствующий цвет, а её название появится в списке справа.

    Чтобы удалить страну из списка, нажмите на красную корзину рядом с ней.

    Чтобы сбросить все метки, нажмите кнопку "Сбросить все метки" под картой.

💻 Технологии

    Leaflet — отрисовка карты и работа с GeoJSON.

    Bootstrap 5 — стили и модальное окно.

    Font Awesome — иконки.

    Natural Earth Data — упрощённые границы стран (через geojson.xyz).

    localStorage — хранение данных в браузере.
