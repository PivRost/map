🗺️ Карта путешествий — страны

Интерактивная веб-карта для отметки посещённых стран и планирования будущих поездок.
Выберите категорию для любой страны, и она закрасится соответствующим цветом. Все данные сохраняются в вашем браузере (localStorage).

Демо: https://pivrost.github.io/map/
✨ Возможности

    🖱️ Клик по стране — открывает модальное окно с выбором категории.

    🎨 4 категории с уникальными цветами:

        ✅ Посетил — зелёный

        ⭐ Хочу посетить — жёлтый

        🛂 Жду визу — оранжевый

        🚫 Вряд ли посещу — красный

    📋 Список отмеченных стран справа с цветными индикаторами.

    🗑️ Удаление отдельной страны прямо из списка.

    🔄 Сброс всех меток одной кнопкой.

    💾 Автосохранение в localStorage — после перезагрузки страницы ваши отметки остаются.

    🔍 Поиск мест на карте (геокодер) — введите название страны или города, и карта плавно переместится к нему.

    🌍 Карта на основе Leaflet с подложкой OpenStreetMap.

    🚀 Работает на GitHub Pages без дополнительных настроек.

🛠️ Технологии

    Leaflet — интерактивная карта

    Leaflet Control Geocoder — поиск мест

    Bootstrap 5 — стили и модальное окно

    Font Awesome — иконки

    Natural Earth Data — упрощённые границы стран (GeoJSON)

🚀 Как развернуть
Вариант 1: GitHub Pages (рекомендуется)

    Создайте новый репозиторий на GitHub.

    Загрузите в корень репозитория файлы index.html и countries.geojson.

    В настройках репозитория (Settings → Pages) выберите ветку main (или master) и папку /root, нажмите Save.

    Через пару минут ваш сайт станет доступен по адресу https://ваш-логин.github.io/имя-репозитория.

Вариант 2: Локальный запуск (для тестирования)

Если вы хотите запустить проект на своём компьютере, используйте любой локальный сервер, чтобы избежать проблем с CORS:

    Python 3: в папке с проектом выполните python -m http.server, затем откройте http://localhost:8000.

    VS Code: установите расширение Live Server и запустите через него.

    Mongoose (Windows): скачайте mongoose.exe, положите в папку и запустите.

📖 Использование

    Откройте страницу с картой.

    Кликните по любой стране на карте — появится модальное окно с выбором категории.

    Выберите одну из четырёх категорий (или нажмите "Убрать метку", чтобы удалить цвет).

    Страна перекрасится в соответствующий цвет, и её название появится в списке справа.

    Чтобы удалить страну из списка, нажмите на красную корзину рядом с ней.

    Чтобы сбросить все метки, нажмите кнопку "Сбросить все метки" под картой.

    Поиск места: в поле поиска в верхнем левом углу карты введите название страны или города. Карта приблизится к выбранному месту и поставит временный маркер.

🧪 Пример данных

Файл countries.geojson содержит упрощённые границы стран в формате GeoJSON. Источник: Natural Earth 110m.
Если вы хотите заменить его на другой набор (например, более детальный), просто подставьте свой файл с тем же именем.
