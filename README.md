🗺️ Карта путешествий (Map)

Интерактивная карта мира для отметки посещённых стран и планирования будущих поездок.
Вы можете присвоить каждой стране одну из четырёх категорий, и она будет закрашена соответствующим цветом. Все данные сохраняются в вашем браузере (localStorage).
✨ Возможности

    🖱️ Клик по стране — открывает модальное окно с выбором категории.

    🎨 4 категории с уникальными цветами:

        ✅ Посетил — зелёный

        ⭐ Хочу посетить — жёлтый

        🛂 Жду визу — оранжевый

        🚫 Вряд ли посещу — красный

    📋 Список отмеченных стран справа с цветными индикаторами.

    🗑️ Удаление отдельной страны прямо из списка.

    🔄 Сброс всех меток одной кнопкой.

    💾 Автоматическое сохранение в localStorage — после перезагрузки страницы все ваши отметки остаются.

    🌍 Карта на основе Leaflet с подложкой OpenStreetMap (названия стран и городов видны всегда).

    📦 Полностью локальная работа — все библиотеки и данные (GeoJSON) лежат рядом с HTML, не требуется интернет для отображения границ (но подложка OSM загружается с сети).

💻 Технологии

    Leaflet — отрисовка карты и работа с GeoJSON.

    Bootstrap 5 — стили и модальное окно.

    Font Awesome — иконки.

    Natural Earth Data — упрощённые границы стран (через geojson.xyz).

    localStorage — хранение данных в браузере.
